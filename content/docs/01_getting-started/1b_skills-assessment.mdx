## Self-Assessment Quiz

Answer honestly:

### Coding Ability

<Checklist>
<ChecklistItem>I can write a complete program from scratch (not just tutorials)</ChecklistItem>
<ChecklistItem>I can debug errors without copying from Stack Overflow</ChecklistItem>
<ChecklistItem>I can read code written by others and understand it</ChecklistItem>
<ChecklistItem>I can write tests for my code</ChecklistItem>
<ChecklistItem>I understand version control (branching, merging, conflicts)</ChecklistItem>
</Checklist>

**0-1 checked:** Focus on learning fundamentals first
**2-3 checked:** Ready for simple contributions
**4-5 checked:** Ready for moderate contributions

### Codebase Navigation

<Checklist>
<ChecklistItem>I can navigate an unfamiliar project structure</ChecklistItem>
<ChecklistItem>I can find where specific functionality lives</ChecklistItem>
<ChecklistItem>I understand common patterns (MVC, service layers, etc)</ChecklistItem>
<ChecklistItem>I can trace execution flow through a codebase</ChecklistItem>
</Checklist>

**0-1 checked:** Spend more time studying codebases
**2-3 checked:** Adequate for getting started
**4 checked:** You're well prepared
