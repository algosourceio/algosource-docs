## Git Configuration

Essential git config for contributors:

```bash
# Set your identity
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Better defaults
git config --global init.defaultBranch main
git config --global pull.rebase true
git config --global fetch.prune true

# Helpful aliases
git config --global alias.st status
git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.lg "log --oneline --graph --all"
```

## VS Code Extensions

Recommended extensions:

- **GitLens** — Supercharged git capabilities
- **GitHub Pull Requests** — Review PRs in VS Code
- **Error Lens** — Inline error display
- **ESLint/Prettier** — Code formatting
- **Remote - Containers** — Develop inside containers

<Callout type="tip">
Most open source projects have recommended extensions. Check for a `.vscode/extensions.json` file.
</Callout>
