# Open Source Programs

Programs that can accelerate your open source journey and provide mentorship, stipends, and recognition.

## Featured Programs

<CardGrid>
<Card title="Google Summer of Code">
**Stipend:** $1,500 - $6,600

A global program where students work with open source organizations on 10-22 week projects.
</Card>

<Card title="Hacktoberfest">
**Reward:** Limited edition swag

Annual event encouraging contributions to open source. Complete 4 quality PRs in October.
</Card>

<Card title="Outreachy">
**Stipend:** $7,000

Paid internships for underrepresented groups in open source and tech.
</Card>

<Card title="LFX Mentorship">
**Stipend:** $3,000 - $6,600

Linux Foundation's mentorship program across their project ecosystem.
</Card>
</CardGrid>

<Callout type="tip" title="Start Early">
Most programs have application periods months before they start. GSoC applications typically open in March for a summer program.
</Callout>

## Program Timeline

| Program | Application Period | Program Duration |
|---------|-------------------|------------------|
| GSoC | March-April | May-August |
| Outreachy | February & August | May & December |
| Hacktoberfest | October | October |
| LFX | Rolling | Varies |
