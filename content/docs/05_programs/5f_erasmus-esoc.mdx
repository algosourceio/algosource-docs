## Program Types

### Erasmus+ Internships
Structured internships with European companies or organizations working on open source.

### European Summer of Code (ESoC)
Community-driven alternative to GSoC in Europe.

## Key Differences

| Aspect | Erasmus+ | ESoC |
|--------|----------|------|
| **Organizer** | European Commission | Community-run |
| **Eligibility** | EU/EEA only, age restricted | EU/EEA, more flexible |
| **Duration** | 2-3 months | 3-4 months |
| **Stipend** | Higher | Medium |
| **Projects** | Broader tech scope | Open source focused |

## Application Timeline

Varies by program - typically:
- Applications: January-March
- Mentee Selection: April-May  
- Program: June-September

## Tips for Success

- Focus on European projects (GNOME, KDE, etc.)
- Learn EU work regulations and labor laws
- Consider visa/travel requirements
- Strong English communication essential
